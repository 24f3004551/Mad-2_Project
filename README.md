# HMS‑V2 (Hospital Management System – v2)

Full‑stack project built with Flask, Vue.js, SQLite, and Redis.

## Folder Structure

- **frontend/** – Vue 3
- **backend/** – Flask API server
- **SQLite database** – for persistent data
- **Redis** – for cache and background jobs

## How to Run

1. `cd backend && source venv/bin/activate && source venv/bin/activate && flask run`
2. `cd frontend && npm install && npm run dev`
