# Hospital_Management_App

This is my MAD-1 Project. A flask-python based hospital management app.

- The app has 3 main roles
  Admin
  Patient
  Doctor

## Admin

Admin can add doctors, delete and blacklist both doctors and patients

Admin can view all appointments and delete cancelled appointments

Admin can create departments and assign doctors to it

## Doctor

Doctors can give treatment to booked appointments

Doctors can cancel appointments and also see patient history

Doctor can setup their available slots and also update them

## Patient

Patient can book appointments with available slots

Patients can see their treatment info
